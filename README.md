# かんばんボードアプリケーション

Vue 3, TypeScript, Viteで構築されたシンプルなかんばんボードアプリケーションです。

## 主な機能

- **タスク管理**: タスクの作成、編集、削除ができます。
- **ステータス管理**: タスクは以下の4つのステータスを持つことができます:
  - 新規
  - 作業中
  - 完了
  - 中止
- **ドラッグ＆ドロップ**: タスクをドラッグ＆ドロップすることで、ステータス間を簡単に移動できます。
- **期日設定**: 各タスクに期日を設定できます。
- **マテリアルデザイン**: UIはGoogleのマテリアルデザイン原則に基づいてスタイリングされています。
- **確認ダイアログ**: タスクを削除する前に確認ダイアログが表示されます。

## 実行方法

1.  **プロジェクトディレクトリに移動します**:
    ```bash
    cd kanban_board
    ```

2.  **依存関係をインストールします**:
    ```bash
    npm install
    ```

3.  **開発サーバーを実行します**:
    ```bash
    npm run dev
    ```

4.  **ブラウザでアプリケーションを開きます**:
    アプリケーションは [http://localhost:5173](http://localhost:5173) で利用できます。

## 単体テスト

このプロジェクトでは、Vitestを使用した単体テストがセットアップされています。
主に `src/store.ts` の状態管理ロジック（タスクの追加、編集、削除など）がテスト対象です。

### テストの実行

以下のコマンドを実行することで、単体テストを実施できます。

```bash
npm test
```

## プロジェクト構成

- `src/components`: アプリケーションのVueコンポーネントが含まれています。
  - `KanbanBoard.vue`: カラムを保持するメインのボードコンポーネントです。
  - `TaskCard.vue`: 個々のタスクカードを表します。
  - `NewTaskModal.vue`: タスクの作成と編集を行うためのモーダルです。
- `src/types.ts`: `Task`と`Status`のTypeScriptの型を定義しています。
- `App.vue`: アプリケーションのルートコンポーネントです。
- `main.ts`: アプリケーションのエントリーポイントです。
